Detailed Description

Class to monitor USB plug/unplug events and detect private key presence.

Class responsible for detecting USB plug/unplug events on Windows.

Class to watch for USB events (plug/unplug).

Creates a hidden window to receive device change notifications from Windows.

Uses a hidden window to listen for system device change notifications. Can identify USB drives containing private keys and notify the GUI.
