\doxysubsection{Detailed Description}
Provides functionality to verify a digitally signed PDF using RSA and SHA-\/256. 

This module extracts the digital signature from a PDF\textquotesingle{}s metadata, reconstructs a clean copy without the signature, generates its hash, and validates the signature using a provided RSA public key.

verify.\+verify\+\_\+pdf (\begin{DoxyParamCaption}\item[{}]{pdf\+\_\+path}{, }\item[{}]{public\+\_\+key\+\_\+path}{}\end{DoxyParamCaption})}



Verifies the digital signature of a PDF file. 

This function reads the embedded signature from the metadata (under {\ttfamily /\+Signature} key), removes the signature from the metadata to obtain a clean version of the PDF, creates a SHA-\/256 hash of this clean copy, and then verifies the hash against the stored signature using the provided RSA public key.
