\doxysubsection{Detailed Description}
GUI frame for generating RSA key pairs and saving them to disk (public) and USB (private). 

Uses customtkinter for UI rendering. Public key is stored in selected folder; private key is saved to USB drive after password-\/based encryption. USB plug detection included.
