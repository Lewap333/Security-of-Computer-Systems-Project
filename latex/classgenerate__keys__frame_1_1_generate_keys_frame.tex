\doxysubsection{Detailed Description}
A customtkinter frame for RSA key pair generation with USB support. 

Provides UI for\+:
\begin{DoxyItemize}
\item selecting public key destination
\item monitoring USB plug/unplug to store private key
\item entering password to encrypt private key
